ML Projects

AI / ML · Academic Mini Project · NMIT

Graph Search Algorithm Optimizer

A Python-based benchmarking system that implements and compares multiple graph search algorithms including BFS, DFS, and heuristic-based approaches. The project constructs diverse graph structures using NetworkX and Pandas, then measures performance metrics such as time complexity, path efficiency, and node traversal counts across each algorithm. Built to answer a genuine question: which search strategy actually performs best on real graph datasets, and under what conditions does one approach outperform another? Output includes structured comparison tables and per-algorithm profiling reports.

Machine Learning · Classification · Business Application

Customer Churn Prediction Model

A supervised learning pipeline built to predict whether a customer will churn, enabling businesses to proactively retain at-risk users. The project covers end-to-end ML workflow: raw data ingestion, exploratory data analysis, feature engineering on usage patterns and behavioral signals, handling class imbalance, and model training using classification algorithms. Evaluation metrics include accuracy, precision, recall, F1-score, and AUC-ROC curve analysis. The model outputs both binary churn predictions and probability scores, making it actionable for retention teams.
